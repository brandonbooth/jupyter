## Start


### Start
Create virtual env
```sh
virtualenv venv
```

Activate virtual env
```sh
source venv/bin/activate
```


Install jupyter
```sh
pip install jupyterlab
```

Once installed, launch JupyterLab with:
```sh
jupyter-lab
```
